# Sistema de Reservas — Peluquería 1

Sistema de reservas para peluquería: landing pública para que los clientes agenden su hora, y panel privado para que los profesionales gestionen sus reservas y horarios.

## Stack técnico

- React
- Vite
- Tailwind CSS
- Node.js
- Express
- Prisma
- PostgreSQL (Supabase)
- Redis (Upstash)
- MongoDB (Atlas)
- JWT
